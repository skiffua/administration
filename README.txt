Адреса https://moodle-ua.free.nf/

Лінк для завантаження https://www.infinityfree.com/

логін root

Хід проведення профілактичних робіт на пк:
1. Видаляємо увесь непотрібний софт, чистка усіх віртуальних машин
2. Перенесення усіх учнівських робіт в архів (\\Komp0\archiv)
3. Створюємо точку відновлення системи - BACKUP_[YEAR] (обов'язково!)
4. Створюємо простого користувача - student (без прав адміністратора - простий користувач)
5. Налаштовуємо заставку робочого столу (нейтральна фрактальна) та блокуємо зміни
6. Сворюємо пароль для адміністратор - Admin[номер-пк^2]?
7. Інвентеризація на інсталяція необхідного софту
8. Запускаємо необхідні скрипти ./скрипти (Squid-проксі обов'язково)
