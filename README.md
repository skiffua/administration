# Паспорт комп'ютерного класу

Даний документ містить перелік програмних засобів для управління класом, проведення тестувань, обміну інформацією та регламент технічного обслуговування.

## 1. Основні програмні засоби

* **Veyon** — система управління комп'ютерним класом.
    * *Інструкція:* `.\veyon.docx`
* **Moodle** — система підтримки навчання.
    * *Адреса:* [moodle-ua.free.nf](https://moodle-ua.free.nf/)
    * *Хостинг:* iFastNet / InfinityFree
    * *Бекапи:* `\\Komp0\archiv\backup`
* **Safe Exam Browser (SEB)** — монопольне проходження тестів.
    * *Завантаження:* [Офіційний сайт](https://safeexambrowser.org/download_en.html)
    * *Скрипти:* Папка `\scripts` (запуск через Veyon)

## 3. Регламент профілактичних робіт
1.  **Чистка:** Видалення зайвого ПЗ, ревізія віртуальних машин.
2.  **Архівація:** Перенесення робіт учнів у `\\Komp0\archiv`.
3.  **Резервне копіювання:** Створення точки відновлення `BACKUP_[YEAR]`.
4.  **Користувачі:** Створення обмеженого запису `student`.
5.  **Інтерфейс:** Встановлення нейтральної заставки (блокування змін).
6.  **Безпека:** Пароль адміністратора за формулою `Admin[номер-пк^2]?`.
7.  **Скрипти:** Обов'язковий запуск `Squid-проксі` з папки `\scripts`.
